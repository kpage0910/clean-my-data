# Data Cleaner

**Data Cleaner** is a simple web tool that helps anyone clean messy CSV files in seconds. Just upload your CSV, and get a cleaned version back—no setup, no coding, no complicated options.

## What does it do?

- Removes duplicate rows
- Fills in missing values
- Cleans up text and column names
- Fixes common data issues automatically
- Gives you a summary of what was fixed

## Who is it for?

- Beginners who want to quickly clean data for projects or homework
- Anyone who doesn’t want to learn Excel formulas or Python
- People who just want their data cleaned—fast and easy

## Tech Stack

- **Backend:** FastAPI (Python), Pandas, PostgreSQL
- **Frontend:** React
- **Authentication:** Google OAuth
- **AI (optional):** OpenAI API, Ollama (Llama 3.2)
- **Deployment:** Vercel/Netlify (frontend), Render/Railway/Heroku (backend)

**Just upload, clean, and download. That’s it!**
